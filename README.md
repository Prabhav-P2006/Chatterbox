# 📱 Chatterbox

**Decentralized Bluetooth Communication**
*Connect without internet, communicate without limits.*

---

## ✨ Features

- 🔐 **Privacy First** – No servers, no data collection.
- 📡 **Offline** – Works entirely through Bluetooth.
- 🌍 **Mesh Networking** – Messages hop through devices.
- 🎨 **Modern UI** – Built with Flutter.

---

## 🚀 Quick Start

```bash
# Install dependencies
flutter pub get

# Run the app
flutter run
```

---

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **Connectivity:** Bluetooth Low Energy (BLE)
